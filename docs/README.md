# sodimo/dotfiles docs

Operator-facing docs for the chezmoi-managed user-scope portion of the
sodimo stack. OS-level and system-scope docs live in `sodimo/harness`;
MCP implementation docs live in `sodimo/mcp`.

## Index

- [kyuz0-toolbox.md](kyuz0-toolbox.md) — How the llama-swap + llama-server
  flag baselines were interpolated from kyuz0/amd-strix-halo-toolboxes,

  divergences we carry from kyuz0's defaults.

- [resync-runbook.md](resync-runbook.md) — Step-by-step checklist for the
  next operator to pull a fresh copy of kyuz0's reference repo and
